# Magical Birthday Site

Open `index.html` in a browser to run the website.

## Quick customization

- Edit `content.js` to change the names, quotes, messages, and pet notes.
- Add photos using the exact filenames already listed in `content.js`.
- Replace the music by dropping your file at `assets/music/birthday-bgm.mp3`.
- The page starts music only after clicking the main start button because browsers block autoplay.

## Photo slots

- `assets/photos/memory-1.jpg`
- `assets/photos/memory-2.jpg`
- `assets/photos/memory-3.jpg`
- `assets/photos/memory-4.jpg`
- `assets/photos/passed-pet.jpg`
- `assets/photos/pet-1.jpg`
- `assets/photos/pet-2.jpg`

If a file is missing, the site shows a styled placeholder instead of breaking.
